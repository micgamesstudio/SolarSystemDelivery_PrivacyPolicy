Privacy Policy

Last updated: February 22, 2026

MicGames Studio built the game Solar System Delivery as a free application. This service is provided as is.

This page informs users regarding policies with the collection, use, and disclosure of information.

1. Information Collection and Use

Solar System Delivery:

  • Does not require user registration
  • Does not collect personal data directly
  • Does not transmit user data to external servers
  • Stores only game progress data locally on the user’s device

Game save data remains stored exclusively on the device and is not shared with third parties.

The app includes a device performance benchmark system that runs locally at startup. Benchmark data:

  • Is not stored
  • Is not transmitted
  • Is not shared

2. Advertising

This app uses Google AdMob, a service provided by Google LLC.

AdMob may automatically collect certain information, including:

  • Advertising ID
  • IP address
  • Device information
  • App usage data

This information is used to provide and improve advertising services.

For more information, please review Google’s Privacy Policy:

https://policies.google.com/privacy

3. Internet Connection

The game works offline. However, an internet connection is required to display advertisements.

4. Children’s Privacy

This app is not specifically directed at children under the age of 13.

The developer does not knowingly collect personal information from children.

5. Data Security

MicGames Studio does not directly collect or store personal data from users.
